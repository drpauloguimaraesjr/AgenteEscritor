🔗 Conexão com o Agente (RAG Local)

Cole a URL do tunnel que aponta para o IPagent. Local: http://localhost:5000

🧠 OpenRouter API (Geração de Texto)

O sistema híbrido usa IPAgent local para busca RAG e OpenRouter para gerar textos com IA de qualidade superior. Aceita pagamento via MetaMask / Crypto. Cada interação gera dados de treinamento para o seu Llama3 local.

Obtenha em openrouter.ai/keys — aceita MetaMask, cartão e crypto

Cada resposta do Claude é automaticamente salva como par de treinamento para o seu Llama3 local. Quando tiver exemplos suficientes, poderá fazer fine-tuning e rodar 100% local.
